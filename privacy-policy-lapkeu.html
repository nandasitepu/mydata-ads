Kebijakan Privasi

Terakhir Diperbarui: 13 Maret 2026

Selamat datang di aplikasi Laporan Keuangan. Kami sangat menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi serta informasi keuangan Anda secara aman dan transparan.

1. Informasi yang Kami Kumpulkan

Dalam menjalankan fungsinya, kami dapat mengumpulkan beberapa jenis informasi:

  • Informasi Identitas: Nama dan alamat email saat pendaftaran akun.
  • Data Transaksi: Catatan pemasukan, pengeluaran, anggaran, dan detail kategori yang Anda input secara manual.
  • Informasi Teknis: Jenis perangkat, versi sistem operasi, dan log aktivitas anonim untuk perbaikan sistem.

2. Penggunaan Data

Data Anda digunakan secara eksklusif untuk kepentingan operasional aplikasi:

  • Menyajikan visualisasi laporan keuangan dan analisis anggaran.
  • Sinkronisasi data antar perangkat melalui layanan cloud terenkripsi.
  • Mengoptimalkan fitur aplikasi berdasarkan pola penggunaan secara agregat (anonim).

3. Keamanan Informasi

Kami menerapkan standar keamanan tinggi untuk melindungi data Anda:

  • Enkripsi End-to-End: Transmisi data antara aplikasi dan server menggunakan protokol SSL/TLS.
  • Penyimpanan Lokal: Opsi untuk menyimpan data hanya di memori perangkat tanpa mengirimkannya ke server.
  • Kerahasiaan: Kami menjamin tidak akan menjual atau membagikan data keuangan spesifik Anda kepada pihak ketiga untuk kepentingan iklan.

4. Kontrol dan Hak Anda

Anda memiliki kendali penuh atas data yang Anda simpan:

  • Edit & Hapus: Anda dapat mengubah atau menghapus data transaksi kapan saja.
  • Penutupan Akun: Anda berhak menghapus akun secara permanen yang akan menghapus seluruh data terkait di server kami.
  • Ekspor: Fitur untuk mengunduh data dalam format PDF atau Excel untuk keperluan pencatatan pribadi.

5. Layanan Pihak Ketiga

Aplikasi ini dapat berinteraksi dengan layanan pihak ketiga (seperti Firebase untuk analitik atau AdMob untuk iklan). Pihak-pihak ini hanya mengakses data teknis non-pribadi sesuai dengan kebijakan privasi mereka masing-masing.
